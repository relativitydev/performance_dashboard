PDB_SelectOldAgentResourcesEDDS.sql - Shows all old GUID resources that still exist within Relativity Instance.  Use this to view Resources to be deleted with the DELETE version of the script.

PDB_SelectOldAgentResourcesWorkspace.sql - Shows all old GUID resources that still exist within Relativity Workspace.  Use this to view Resources to be deleted with the DELETE version of the script.

PDB_DELETEOldAgentResourcesEDDS.sql - Removes all old GUID resources from Relativity Instance

PDB_DELETEOldAgentResourcesWorkspace.sql - Removes all old GUID from current workspace (Make sure to run this scoped in the installed workspace)
